Smart Garden

Uma proposta inovadora que te permite monitorar o seu jardim e suas plantas de qualquer lugar do mundo.
Cadastrar-se


timeline

Gráficos Dinâmicos

Monitore dados climáticos como; umidade do ar, temperatura ambiente, umidade do solo e temperatura do solo através de graficos dinâmicos, com o uso dos graficos fica muito mais fácil perceber variações nesses valores e assim tomar a melhor decisão para o seu jardim, é possivel fazer o download desses dados através de relatorios diarios, semanais e mensais.

android

Plataforma responsiva

O framework utilizado é o Materialize CSS criado por uma equipe de estudantes da Carnegie Mellon University - USA, baseado no Material Design desenvolvido pela Google, graças a isso esta webapp possui design responsivo para os variados tamanhos de telas, seu design lembra muito o sistema operacional android, não é pra menos pois a intenção é desenvolver um sistema de design que permite unificar a experiência do usuário em qualquer plataforma.
